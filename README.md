# 網頁五子棋對戰遊戲

一個支持區網對戰的網頁五子棋遊戲，使用 WebSocket 實現實時通信。

## 功能特色

- 🎮 **房間系統**：玩家可以創建或加入房間進行對戰
- 🎲 **隨機先手**：每局遊戲隨機決定先手玩家
- ⚡ **實時對戰**：使用 WebSocket 實現即時通信
- 🏆 **自動判勝**：自動檢測五子連線勝利條件
- 🔄 **重新開始**：支持遊戲結束後重新開始
- 📱 **響應式設計**：支持手機和電腦端

## 技術棧

- **前端**：HTML5 + CSS3 + JavaScript + Canvas
- **後端**：Node.js + Express + Socket.io
- **通信**：WebSocket 實時雙向通信
